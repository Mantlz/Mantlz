# Welcome

Mantlz is a powerful form solution platform that helps you create beautiful, customizable forms for your website or application. Whether you need feedback forms, contact forms, or waitlist forms, Mantlz provides the tools to collect and manage user submissions efficiently.

## What is Mantlz?

Mantlz is a complete form solution platform that enables developers and businesses to:

- **Create customizable forms** with minimal code
- **Collect user feedback** through star ratings and comments
- **Build contact forms** to gather inquiries and questions
- **Generate waitlists** for upcoming products or services
- **Analyze form data** with built-in analytics

Our SDK offers extensive customization options, dark mode support, and seamless integration with Next.js and other frameworks.

## Key Features

### Multiple Form Types

Mantlz provides three main form components:

- **FeedbackForm** - Collect user ratings and comments
- **ContactForm** - Gather contact information and messages
- **WaitlistForm** - Build an early access waitlist with counters

### Extensive Customization

All forms come with extensive styling options:

- **Preset themes** including default, dark, purple, neobrutalism, and more
- **Appearance API** for complete control over styling
- **Tailwind CSS compatibility** for seamless integration
- **Dark mode support** across all components

### Developer Experience

Mantlz is built with developers in mind:

- **Simple API** that's easy to implement
- **Comprehensive documentation** for all features
- **Powerful SDK** for Next.js and React
- **REST API** for custom integrations

### Data Management

Take control of your form submissions:

- **Submission tracking** in the Mantlz dashboard
- **Analytics** to understand user behavior
- **Email notifications** for new submissions
- **Export options** for your data

## Who Should Use Mantlz?

Mantlz is perfect for:

- **Developers** building websites and applications
- **Startups** collecting early user feedback
- **SaaS companies** creating waitlists for new features
- **Businesses** looking for customizable contact forms
- **Product teams** gathering customer insights

## Example Form

Here's a simple example of how to add a feedback form to your Next.js application:

```tsx
import { FeedbackForm } from '@mantlz/nextjs';

export default function FeedbackPage() {
  return (
    <div className="container mx-auto py-8">
      <h1 className="text-2xl font-bold mb-4">We'd Love Your Feedback</h1>
      
      <FeedbackForm 
        formId="your-form-id"
        title="How was your experience?"
        description="Your feedback helps us improve"
        theme="purple"
      />
    </div>
  );
}
```
