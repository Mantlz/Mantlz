# WaitlistForm Component

The `WaitlistForm` component allows you to create elegant waitlist forms to collect email addresses from users interested in your product or service.

## Basic Usage

```tsx
import { WaitlistForm } from 'mantlz-sdk';

export default function WaitlistPage() {
  return (
    <div className="container mx-auto p-4">
      <WaitlistForm 
        formId="your-waitlist-id"
        title="Join Our Waitlist"
        description="Sign up to get early access to our new product."
      />
    </div>
  );
}
```

## Users Joined Counter

WaitlistForm can display a counter showing how many users have already joined:

```tsx
<WaitlistForm
  formId="your-waitlist-id"
  title="Join Our Waitlist"
  description="Be among the first to try our platform"
  showUsersJoined={true}
  usersJoinedLabel="people have already joined"
/>
```

## Customization Options

### Preset Themes

```tsx
import { WaitlistForm, WAITLIST_THEMES } from 'mantlz-sdk';

// Choose from various built-in themes
<WaitlistForm
  formId="your-waitlist-id"
  theme={WAITLIST_THEMES.DARK} // Options: DEFAULT, DARK, PURPLE, NEOBRUTALISM
/>
```

### Simple Props

```tsx
<WaitlistForm
  formId="your-waitlist-id"
  // Form text content
  title="Join Our Private Beta"
  description="Get early access to our AI-powered platform"
  submitButtonText="Join Waitlist"
  emailPlaceholder="Enter your email address"
  successMessage="You're on the list! We'll be in touch soon."
  
  // Visual styling
  primaryColor="purple"
  borderRadius="lg"
  shadow="md"
  
  // Force dark mode
  darkMode={true}
/>
```

### Advanced Appearance API

```tsx
<WaitlistForm
  formId="your-waitlist-id"
  appearance={(theme) => ({
    baseStyle: {
      container: `bg-${theme === 'dark' ? 'zinc-900' : 'white'} p-6 rounded-xl shadow-lg`,
      form: 'space-y-6',
    },
    elements: {
      input: {
        wrapper: 'relative',
        input: theme === 'dark' 
          ? 'bg-zinc-800 border-zinc-700 text-white' 
          : 'bg-gray-50 border-gray-200',
      },
      submitButton: 'bg-blue-600 hover:bg-blue-700 text-white rounded-lg',
      usersJoinedCounter: theme === 'dark'
        ? 'text-white bg-zinc-800 border border-zinc-700'
        : 'text-gray-700 bg-gray-50 border border-gray-200',
    },
    typography: {
      title: 'text-2xl font-bold',
      description: 'text-gray-500',
      emailPlaceholder: "Your email address",
      submitButtonText: "Join Now",
      usersJoinedText: "people on the waitlist",
    }
  })}
/>
```

## Simplified Appearance API

WaitlistForm also supports the simplified appearance API:

```tsx
<WaitlistForm
  formId="your-waitlist-id"
  appearance={{
    // Direct styling
    container: 'bg-blue-50 p-8 rounded-xl',
    input: 'border-blue-300 focus:ring-blue-500',
    button: 'bg-blue-600 hover:bg-blue-700 text-white',
    counter: 'bg-blue-100 border border-blue-200 text-blue-800',
    
    // Text customization
    submitText: 'Join the Waitlist',
    placeholderText: 'Enter your email to join',
  }}
/>
```

## Post-Submission Redirects

After form submission, you can specify a custom redirect URL (STANDARD and PRO plans only):

```tsx
<WaitlistForm
  formId="your-waitlist-id"
  redirectUrl="/thank-you" // Custom redirect URL (premium feature)
/>
```

## Event Handlers

```tsx
<WaitlistForm
  formId="your-waitlist-id"
  onSubmitSuccess={(data) => {
    // Handle successful submission
    console.log('Email:', data.email);
    analytics.track('Waitlist Joined', { email: data.email });
  }}
  onSubmitError={(error) => {
    // Handle submission error
    console.error('Submission failed:', error);
  }}
/>
```

## Available Props

| Prop | Type | Description |
|------|------|-------------|
| `formId` | `string` | **Required**. The ID of the form in your Mantlz dashboard. |
| `theme` | `string` | The theme to use. Default: `'default'` |
| `appearance` | `object \| function` | Custom styles to apply to the form. |
| `darkMode` | `boolean` | Force dark mode. Default: `undefined` (auto-detect) |
| `primaryColor` | `string` | Primary color theme. Example: `'blue'` |
| `borderRadius` | `string` | Border radius size. Example: `'lg'` |
| `shadow` | `string` | Shadow size. Example: `'md'` |
| `title` | `string` | Form title text. Default: `'Join our waitlist'` |
| `description` | `string` | Form description text. |
| `submitButtonText` | `string` | Text for submit button. Default: `'Join Waitlist'` |
| `emailPlaceholder` | `string` | Placeholder for email field. |
| `successMessage` | `string` | Message shown after successful submission. |
| `showUsersJoined` | `boolean` | Show the number of users who have joined. Default: `false` |
| `usersJoinedLabel` | `string` | Text shown next to the users joined count. |
| `onSubmitSuccess` | `function` | Callback fired on successful submission. |
| `onSubmitError` | `function` | Callback fired on submission error. |
| `redirectUrl` | `string` | URL to redirect to after submission (premium). |
